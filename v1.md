# V1



## Différents fichiers

### minos.py
- but : gérer les objets minos, leur construction et toutes leur vie, les afficher
- contient : la classe Mino qui gère tous les attributs et les méthodes
- Fonctionnement : Pour lors de la création, chaque mino prend en paramètre l'écran sur lequel il devra s'afficher ainsi que ses dimensions.
 
### draw.py
- Gerer les affichages de toute la page pygame
- Fonctionnement : Affiche le fond, définit les dimensions etc...
  
### graph.py
- but : Génerer les graphs à partir de fichiers csv remplis durant la simulation

### files.py
- but : Gérer la création et le chargment des fichiers csv
- fonctionnement : fonctions appellées ponctuellement pour save ou load les fichiers csv

### main.py
- but : coordonner les autres fichiers afin d'optenir un rendu
- Créé l'écran qui sera une variable globale

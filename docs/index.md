# LifeSim 

[![Version](https://img.shields.io/badge/version-2.0.0-blue.svg)](https://github.com/maxence-dev1/LifeSim)
[![Python: 3.9+](https://img.shields.io/badge/python-3.9+-green.svg)](https://www.python.org/)

**LifeSim** est un logiciel de simulation de vie de petits êtres (**Minos**) en fonction de certains paramètres.

---


## Fonctionnalités Clés

* **Génétique aléatoire** : répartition des atttributs selon une loi de Gauss
* **Minos intelligents** : Les minos possèdent une intelligence
